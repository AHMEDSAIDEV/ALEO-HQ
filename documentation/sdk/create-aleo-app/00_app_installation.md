## Scaffolding Your First Aleo Project

:::note
**Compatibility:**
[Node.js](https://nodejs.org/en/) version 18+
:::

With NPM:

```bash
npm create aleo-app@latest
```

Then follow the prompts!

Currently supported template presets include:
- `react` + `javascript`
- `react` + `javascript` + `leo`
- `node` + `javascript`
- `vanilla (javascript)`

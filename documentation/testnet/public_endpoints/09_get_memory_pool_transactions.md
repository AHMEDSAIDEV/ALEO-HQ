```bash title=ENDPOINT
GET /testnet3/memoryPool/transactions
```

Returns the transactions in the memory pool.

### Arguments

None

### Response

| Parameter |                    Type                    |        Description        |
|:---------:|:------------------------------------------:|:-------------------------:|
| `result`  | [array](../../concepts/03_transactions.md) | The array of transactions |

```bash title=ENDPOINT
GET /testnet3/peers/count
```

Returns the number of peers connected to the node.

### Arguments

None

### Response

| Parameter |  Type  |                Description                |
|:---------:|:------:|:-----------------------------------------:|
| `result`  | number | The number of peers connected to the node |

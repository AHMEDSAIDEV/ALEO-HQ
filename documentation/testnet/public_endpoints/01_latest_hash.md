```bash title=ENDPOINT
GET /testnet3/latest/hash
```

Returns the latest block hash.

### Arguments

None

### Response

| Parameter |  Type  |                 Description                  |
|:---------:|:------:|:--------------------------------------------:|
| `result`  | string |   The block hash of the most recent block    |

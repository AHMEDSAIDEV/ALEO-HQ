## `leo init`

To initialize a Leo project in an existing directory, run:
```bash
leo init
```
This will initialize the current directory with the same package directory setup.

### Optional Flags
`leo init` supports the same flags as `leo new`
```bash
leo init --bin
```
```bash
leo init --lib
```
